# Vue 项目打包工具流程
## VueProjectBuilder 类概述
**作用**: 将 AI 生成的 Vue 项目源码打包构建，生成可部署的静态文件（dist 目录）


## 核心方法

| 方法 | 说明 |
|------|------|
| `buildProjectAsync(path)` | 异步构建，使用虚拟线程，不阻塞主线程 |
| `buildProject(path)` | 同步构建，返回是否成功 |


## 工作流程

```
校验项目目录 → 检查 package.json → npm install → npm run build → 验证 dist 目录
```

## Java 执行终端命令

### 调用链路

```
executeCommand() → RuntimeUtil.exec() → Runtime.getRuntime().exec() → Process
```
其中RuntimeUtil.exec() 是HuTool封装的shell命令执行工具
